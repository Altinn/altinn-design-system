# Altinns interne komponentbibliotek

## Hvordan installere

For å legge til komponentbiblioteket i ditt prosjekt, naviger til mappen hvor `package.json`-filen befinner seg og kjør en av følgende kommandoer:

### NPM

```
npm install @altinn/altinn-design-system
```

### Yarn

```
yarn add @altinn/altinn-design-system
```

## Hvordan bruke Storybook

På de fleste komponentene ligger det en meny øverst til venstre med valgene `Canvas` og `Docs`.

- `Canvas`: Viser den spesifikke varianten av komponenten som er valgt.
- `Docs`: Viser dokumentasjon generelt til komponenten og ulike måter den kan brukes på.
